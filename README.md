# Next.js & wasm

## 事前設定

- npmパッケージのインストール  
`yarn`

## Rustでwasmのコンパイル

- Rustに追加インストールが必要なパッケージ  
`cargo install wasm-pack`
- Rustのbuild  
`yarn wasm`

## Next.jsの起動

- dev起動  
`yarn run`
- Build  
`yarn build`
